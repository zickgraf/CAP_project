13 Dummy categories

A dummy category pretends to support certain CAP operations but has not actual implementation. This is useful for testing or compiling against a certain set of CAP operations.

13.2 Constructors

13.2-1 DummyCategory
‣ DummyCategory( options )( operation )

Returns: a category

Creates a dummy category subject to the options given via options, which is a record passed on to CategoryConstructor (11.2-1). Note that the options {category,object,morphism}_filter will be set to IsDummyCategory{,Object,Morphism} and create_func_* will be set to dummy implementations (throwing errors when actually called). The dummy category will pretend to support empty limits by default.
